252/*!
253 By default, in <b>InitMULTI</b> (Which is mass initialization of up to 64 <b>DO</b> (Digital Output)) and <b>SetStatusMULTI</b> (Which is mass status set of up to 64 <b>DO</b> (Digital Output)) commands, 8 bytes which is 64 bits (which means 64 <b>DO</b> (Digital Output)) is used in these commands\n
254 This value is set here as <b>FH_DO_MULTI_MaxPeripheralNumberLimit</b>\n
255 On the other hand, <b>FH_DO_MaxPeripheralNumber</b> is the maximum number of the <b>FH_DO</b> (Digital Output)\n
256
257 @warning
258 To use <b>InitMULTI</b> (Which is mass initialization of up to 64 <b>DO</b> (Digital Output)) and <b>SetStatusMULTI</b> (Which is mass status set of up to 64 <b>DO</b> (Digital Output)) commands, <b>FH_DO_MaxPeripheralNumber</b> and <b>FH_DO_MULTI_MaxPeripheralNumberLimit</b> shall be set to 64\n
259*/
260#define FH_DO_MULTI_MaxPeripheralNumberLimit 64
