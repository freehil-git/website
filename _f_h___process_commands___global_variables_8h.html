volatile uint8_t FH_ProcessCommands_ReceiveBuf[]
extern

exteeeeennnn

FH user shall use FH_ProcessCommands_ReceiveBuf in the receive ISR (Interrupt Service Routine) of the Communication Dedicated UART in a separate c file
For this to happen, FH user shall just include FH_ProcessCommands_GlobalVariables.h there
In other word, FH_ProcessCommands_ReceiveBuf array shall receive every byte of the received data there
It is configurable in FH_ProcessCommands_Configs.h
