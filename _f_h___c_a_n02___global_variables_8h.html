◆ FH_CAN02_ReceiveBuf

volatile uint8_t FH_CAN02_ReceiveBuf[]
extern

FH user shall use FH_CAN02_ReceiveBuf in the receive ISR (Interrupt Service Routine) of the CAN 02 in a separate c file
For this to happen, FH user shall just include FH_CAN02_GlobalVariables.h there
In other word, FH_CAN02_ReceiveBuf array shall receive every byte of the received data there
It is configurable in FH_CAN02_Configs.h
