◆ FH_UART03_ReceiveBuf

volatile uint8_t FH_UART03_ReceiveBuf[]
extern

FH user shall use FH_UART03_ReceiveBuf in the receive ISR (Interrupt Service Routine) of the UART 03 in a separate c file
For this to happen, FH user shall just include FH_UART03_GlobalVariables.h there
In other word, FH_UART03_ReceiveBuf array shall receive every byte of the received data there
It is configurable in FH_UART03_Configs.h
